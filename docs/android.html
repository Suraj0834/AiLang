Android SDK Documentation

Complete guide to integrating AiLang into your Android application.

Installation

Add AiLang to your project's build.gradle.kts or build.gradle:

Kotlin DSL (build.gradle.kts)

dependencies {
    implementation("io.github.ailang:ailang-android:1.0.0")
}

Groovy DSL (build.gradle)

dependencies {
    implementation 'io.github.ailang:ailang-android:1.0.0'
}

Initialization

Initialize AiLang in your Application class:

class MyApp : Application() {
    override fun onCreate() {
        super.onCreate()
        
        // Basic initialization
        AiLang.init(this, "YOUR_GEMINI_API_KEY")
        
        // Or with custom configuration
        AiLang.init(
            context = this,
            apiKey = "YOUR_GEMINI_API_KEY",
            config = AiLangConfig.builder()
                .defaultLanguage("en")
                .cacheEnabled(true)
                .cacheDuration(24 * 60 * 60 * 1000L) // 24 hours
                .debugMode(BuildConfig.DEBUG)
                .build()
        )
    }
}

Basic Usage

Use the t() function to translate strings:

// Simple translation
val greeting = AiLang.t("hello")  // Returns translated "Hello"

// In a TextView
textView.text = AiLang.t("welcome_message")

// In Jetpack Compose
Text(text = AiLang.t("title"))

Translation

AiLang provides several ways to translate text:

// Basic translation
val text = AiLang.t("greeting")

// With fallback key
val text = AiLang.t("unknown_key")  // Returns "unknown_key" if not found

Parameters

Dynamic values can be inserted into translations:

// Define in your strings: "Hello, {name}! You have {count} messages."

val message = AiLang.t("welcome", mapOf(
    "name" to userName,
    "count" to messageCount
))

// Result: "¬°Hola, Juan! Tienes 5 mensajes." (in Spanish)

Pluralization

Handle plural forms automatically:

// Define: "{count} item|{count} items"

val itemText = AiLang.t("items", 1)  // "1 item"
val itemsText = AiLang.t("items", 5) // "5 items"

Language Switching

Change language at runtime:

// Change language
AiLang.setLanguage("es")  // Switch to Spanish
AiLang.setLanguage("hi")  // Switch to Hindi

// Get current language
val currentLang = AiLang.getCurrentLanguage()

// Get all supported languages
val languages = AiLang.getSupportedLanguages()
languages.forEach { lang ->
    println("${lang.name} (${lang.code}) - ${lang.nativeName}")
}

// Listen for language changes
AiLang.addLanguageChangeListener { newLang ->
    // Update UI
    recreate()
}

Caching

Translations are cached automatically for performance:

// Clear all cached translations
AiLang.clearCache()

// Configure caching in initialization
AiLangConfig.builder()
    .cacheEnabled(true)
    .cacheDuration(48 * 60 * 60 * 1000L) // 48 hours
    .maxCacheSize(20) // 20 MB
    .build()

Preloading Languages

Pre-translate all strings for offline use:

// Preload a language
AiLang.preloadLanguage("es") { success ->
    if (success) {
        println("Spanish translations preloaded")
    }
}

// Preload multiple languages
listOf("es", "fr", "de").forEach { lang ->
    AiLang.preloadLanguage(lang)
}

RTL Support

Handle right-to-left languages:

// Check if current language is RTL
if (AiLang.isRTL()) {
    // Adjust layout for RTL
    layoutDirection = View.LAYOUT_DIRECTION_RTL
}

// RTL languages: Arabic, Hebrew, Persian, Urdu

Configuration Options

Option Type Default Description
defaultLanguage String "en" Fallback language code
cacheEnabled Boolean true Enable translation caching
cacheDuration Long 86400000 Cache TTL in milliseconds (24h)
maxCacheSize Int 10 Maximum cache size in MB
debugMode Boolean false Enable debug logging
batchSize Int 50 Strings per API call
timeout Int 30 API timeout in seconds
retryCount Int 3 API retry attempts
