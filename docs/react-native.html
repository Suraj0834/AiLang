React Native SDK Documentation

Complete guide to integrating AiLang into your React Native application.

Installation

Using npm

npm install ailang-react-native

Using Yarn

yarn add ailang-react-native

Peer Dependencies

npm install @react-native-async-storage/async-storage

iOS Setup

cd ios && pod install

Setup

Wrap your app with the AiLangProvider:

import React from 'react';
import { AiLangProvider } from 'ailang-react-native';
import App from './App';

const Root = () => (
  <AiLangProvider
    apiKey="YOUR_GEMINI_API_KEY"
    defaultLanguage="en"
    enableCache={true}
  >
    <App />
  </AiLangProvider>
);

export default Root;

Basic Usage

Use the useAiLang hook in any component:

import React from 'react';
import { View, Text, Button } from 'react-native';
import { useAiLang } from 'ailang-react-native';

const MyComponent = () => {
  const { t, setLanguage, currentLanguage } = useAiLang();

  return (
    <View>
      <Text>{t('hello')}</Text>
      <Text>Current: {currentLanguage}</Text>
      <Button title="Switch to Spanish" onPress={() => setLanguage('es')} />
    </View>
  );
};

useAiLang Hook

The hook provides all translation functionality:

const {
  // Translation function
  t,                    // (key: string, params?: object) => string
  
  // Language management
  currentLanguage,      // Current language code
  setLanguage,          // (code: string) => Promise<void>
  supportedLanguages,   // Language[] - all available languages
  isRTL,                // boolean - is current language RTL?
  
  // State
  isLoading,            // boolean - translation in progress
  
  // Cache
  clearCache,           // () => Promise<void>
  preloadLanguage,      // (code: string) => Promise<boolean>
} = useAiLang();

Translation

Simple Translation

const greeting = t('hello');
// Returns: "Hola" (if Spanish)

With Parameters

// String: "Welcome, {name}! You have {count} notifications."
const message = t('welcome', { name: 'John', count: 5 });
// Returns: "¬°Bienvenido, John! Tienes 5 notificaciones."

With Pluralization

// String: "{count} item|{count} items"
const single = t('items', { count: 1 }); // "1 item"
const plural = t('items', { count: 5 }); // "5 items"

Language Switching

import { useAiLang } from 'ailang-react-native';
import { Picker } from '@react-native-picker/picker';

const LanguageSwitcher = () => {
  const { currentLanguage, setLanguage, supportedLanguages } = useAiLang();

  return (
    <Picker
      selectedValue={currentLanguage}
      onValueChange={(lang) => setLanguage(lang)}
    >
      {supportedLanguages.map((lang) => (
        <Picker.Item
          key={lang.code}
          label={lang.nativeName}
          value={lang.code}
        />
      ))}
    </Picker>
  );
};

Batch Translation

Translate multiple strings efficiently:

import { TranslationService } from 'ailang-react-native';

const translationService = new TranslationService({ apiKey: 'YOUR_KEY' });

// Translate multiple texts at once
const results = await translationService.translateBatch(
  ['Hello', 'Goodbye', 'Thank you'],
  'es'
);

// Results:
// [
//   { success: true, translatedText: 'Hola' },
//   { success: true, translatedText: 'Adi√≥s' },
//   { success: true, translatedText: 'Gracias' }
// ]

Caching

Translations are cached automatically using AsyncStorage:

const { clearCache, preloadLanguage } = useAiLang();

// Clear all cached translations
await clearCache();

// Preload translations for offline use
const success = await preloadLanguage('es');
if (success) {
  console.log('Spanish translations cached');
}

Configuration Options

AiLangProvider Props

Prop Type Default Description
apiKey string required Gemini API key
defaultLanguage string "en" Default language code
enableCache boolean true Enable caching
maxCacheSize number 500 Maximum cache entries
cacheDuration number 86400000 Cache TTL (24h in ms)
debugMode boolean false Enable debug logs
